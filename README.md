<h1>🪨📄✂️ Jokenpô</h1>

<h2>Projeto desenvolvido com HTML, CSS e JavaScript que permite ao usuário jogar Pedra, Papel ou Tesoura contra o computador.

O sistema realiza escolhas aleatórias para o computador e exibe o resultado da partida na tela, podendo ser vitória, derrota ou empate.</h2>

<br>

<h1>🚀 Funcionalidades</h1>

<h2>- 🎮 Jogo interativo contra o computador
  <br>
- 🪨 Escolha entre Pedra, Papel ou Tesoura
  <br>
- 🤖 Escolha aleatória do computador
  <br>
- 🏆 Exibição do resultado da partida
  <br>
- 🎨 Interface simples e responsiva
  <br>
- 🖱️ Interação através de botões e eventos</h2>

<br>

<h1>🛠️ Tecnologias utilizadas</h1>

<h2>- HTML5
  <br>
- CSS3
  <br>
- JavaScript</h2>
